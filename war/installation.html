Installation

There are several ways to install the software.

  1. Greasemonkey Script

    Installation
    If you are running Firefox with the Greasemonkey extension installed, you can use this link to install it as a user script. You will be prompted one or two times to grant permission for the script to be installed.
    Disable for a page
    Greasemonkey can be disabled for a particular page by unchecking "Enabled" in the menu Tools > Greasemonkey > Enabled; after disabling the script you'll have to refresh the page to restore its original structure
    Uninstallation
    The script can be uninstalled by selecting "Remove" in the "User Scripts" section of the Tools > Add-Ons panel

  2. Tampermonkey Script

    Installation
    If you are running Chrome with the Tampermonkey extension installed, you can use this link to install it as a user script. You will be prompted one or two times to grant permission for the script to be installed.
    Disable for a page
    Tampermonkey can be disabled for a particular page by unchecking "Enabled" in the Tampermonkey menu; after disabling the script you'll have to refresh the page to restore its original structure
    Uninstallation
    The script can be uninstalled by selecting the "Delete" Action for the script in Tampermonkey's Dashboard (Tampermonkey menu > Dashboard)

  3. Firefox Extension

    Installation
    If you are running Firefox, you can use this link to download it as a standalone Firefox extension. To install the extension, open the file you downloaded with Firefox by using the File > Open File menu (the standard keyboard shortcut is Ctrl+O). You will be prompted one or two times to grant permission for the script to be installed. After installation, Firefox will prompt you to restart before the extension can be used.
    Disable for a page
    The extension includes a keyboard shortcut Ctrl+Alt+W which will disable its effect; after disabling the extension you'll have to refresh the page to restore its original structure. You can also disable the extension by unchecking "Enabled" in the menu Tools > Greasemonkey > Enabled.
    Uninstallation
    The extension can be uninstalled in the Tools > Add-Ons panel by using the "Remove" buttons for both the "Greasemonkey" and the "wrapup-access-scripts" extensions.
    Note on Greasemonkey
    The standalone extension is implemented by packaging the user script alongside Greasemonkey itself

  4. Proxy

    Installation
    You can point any browser at the form on this page. Paste the URL of the page you want to view into the form, and press the "Go" button.
    Disable for a page
    All pages reached through the proxy will be re-written so that all links will continue through the proxy, but browsing in a new or different window will not be affected.
    Limitation
    The proxy can only visit pages that are not secured with SSL, i.e. only URLs that start with http:// and not https://
